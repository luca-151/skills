# Copywriting Line Rules (C1–C5)

Line-level copywriting rules from the sample-messaging passes. These operate below the
document-discipline rules in `references/messaging-doc-discipline.md`: those govern whether a
claim may be made and how it is sourced; these govern whether the line that carries it is
grammatical, load-bearing, and finished. Apply on every original brand line before it ships.

Source: Every Agent messaging framework session, July 13, 2026 (product referred to as
"@Every" / "the Every Agent," formerly Plus One).

---

- **C1. NO IMITATION COMPRESSION.** Short-sentence, fragment-heavy "iconic" register (Apple, Palantir) is earned by a load-bearing fact under the line — a spec, a measurement, an institutional truth. Compression without a spec underneath reads as costume ("One mind. Yours." / "March remembers. June inherits."). Before shipping any compressed line, name the concrete fact it stands on; if there isn't one, write the plain claim instead.

- **C2. EVERY PRONOUN NEEDS A VISIBLE ANTECEDENT — READ THE LINE LITERALLY BEFORE SHIPPING.** "Now installed at yours" (yours what?) and "now it can run yours" (your Every?) shipped twice in one session. QC pass on every line: read it as its literal grammar with no charity. If a possessive or pronoun points at nothing, or the parallel structure breaks when expanded, the line is broken regardless of rhythm.

- **C3. THE LINE MUST SURVIVE ITS OWN LOGIC.** "Your company knows more than anyone in it. @Every is where that becomes true" — the first clause is already true, so the second is nonsense. Test each line's claim as a proposition: is it already true (twist collapses), false (claim fails), or made true/usable by the product (line works)? Only the third ships.

- **C4. GENERATE FOR A KILL RATE, NOT A FINISH LINE.** Model-drafted copy converges on competent-B2B; it does not converge on iconic with more attempts. The working division of labor: the model produces the verified message inventory and volume drafts; the human's taste performs the distillation. Never present a first pass of original brand copy as finished; present it as material with the weak lines already flagged.

- **C5. STATE THE CLAIM PLAIN BEFORE STYLING IT.** For every message, write the unstyled proposition first ("this agent runs Every's company; you can install it in yours"). Style only after the plain version is true, grammatical, and specific. If styling breaks any of the three, ship the plain version.
