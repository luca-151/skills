# Human acceptance — remix-graphic-ad-from-reference

Show the reviewer the **reference** and the **remixed ad** side by side. Ask:

1. **Same ad?** Does the remix read as the same layout/composition/energy as the reference
   (not a generic redesign)? It should feel like "that ad, but for my product."
2. **Product right?** Is it the correct SKU, label legible, logo undistorted, on-brand colors?
3. **Copy right?** Headline, callouts, social proof, and **discount/price** all correct and
   correctly spelled? (Zoom to 100% — text errors hide in thumbnails.)
4. **Brand-safe?** No fear framing, no competitor jab, no unverifiable claim introduced by the swap.
5. **Editable?** If copy/price will change often, was the HTML path used (so edits don't need a re-roll)?

Reject if the layout drifted from the reference, the label is wrong, any text is misspelled, or
the discount value is off.
